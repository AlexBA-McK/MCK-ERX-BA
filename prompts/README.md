# Prompts

This directory contains reusable prompt templates for recurring BA/PO tasks on the McKesson ERX project. Templates use `{{double_brace}}` placeholders for variables you fill in before use.

---

## Available Templates

| File | Purpose |
|---|---|
| [gap-analysis-template.md](./gap-analysis-template.md) | Identify gaps between current and desired state in any project artifact |
| [requirement-clarification-template.md](./requirement-clarification-template.md) | Generate clarifying questions for ambiguous or incomplete requirements |
| [sprint-review-summary-template.md](./sprint-review-summary-template.md) | Summarize sprint outcomes, completed stories, and carry-over items |
| [stakeholder-update-template.md](./stakeholder-update-template.md) | Draft stakeholder update communications from project status |

---

## How to Use a Template

1. Open the template file.
2. Copy the prompt text from the **Template** section.
3. Replace all `{{placeholder}}` variables with your actual values.
4. Paste the completed prompt into GitHub Copilot Chat or your AI assistant.
