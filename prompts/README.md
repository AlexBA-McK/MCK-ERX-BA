# Prompts

This directory contains reusable prompt templates for common BA/PO tasks on the EPAM-McKesson eRx project.

---

## What is a Prompt Template?

A prompt template is a ready-to-use text block you can copy into Copilot Chat, ChatGPT, or any LLM tool. Templates use `{{placeholder}}` syntax for variable parts — replace the placeholders with your specific content before submitting.

Unlike agents, prompt templates are lightweight: no system prompt setup required. Just fill in the blanks and go.

---

## How to Use a Template

1. Open the template file.
2. Read the **Purpose** and **Placeholders** sections in the file header.
3. Copy the prompt text.
4. Replace every `{{placeholder}}` with your specific content.
5. Submit in Copilot Chat or your LLM tool of choice.
